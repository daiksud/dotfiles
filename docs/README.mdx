# 🏠 dotfiles

macOS / Ubuntu (Codespaces) で統一された開発環境を再現するための dotfiles リポジトリです。

[クイックスタート →](./guides/01-quick-start.md)
[GitHub](https://github.com/daiksud/dotfiles)

## 特徴

- **ワンコマンドセットアップ** — `install.sh` を実行するだけで、シェル・エディタ・ツール群がすべて揃う
- **JSON マッピング** — `install_map.json` でシンボリックリンク先を柔軟に制御
- **マルチプラットフォーム** — macOS (Apple Silicon) と Ubuntu (Codespaces) の両方に対応
- **自動 Git ID** — `gh-config-dir.zsh` がリポジトリごとに GitHub アカウントと SSH 署名鍵を切り替え
