# Issue 24: ggrコマンドに引数サポートと自動Acceptを追加

## 日付

2026-04-20

## 背景・目的

`ggr`（ghqリポジトリへの移動コマンド）を引数なしで呼び出す場合はfzfが開くだけだったが、引数を指定した際にそれをfzfの初期検索ワードとして使えるようにしたい。また、結果が1件のみの場合は自動的にAcceptすることで操作を省略したい。

## 変更内容

- `dotfiles/.zsh/go-to-ghq-repository.zsh`
  - `query` の初期値を `${1:-${LBUFFER:-}}` に変更し、引数があればそれをfzfのクエリに使用するようにした
  - fzfに `--select-1` オプションを追加し、候補が1件のみの場合は自動的に選択・Acceptするようにした

## 備考

- zleウィジェット（`^]`キーバインド）経由で呼ばれる場合は引数なしのため、従来通り`LBUFFER`がクエリとして使われる
- `--select-1` はfzfの組み込みオプション
